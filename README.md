

## Installation & Project Setup

1. Clone project
2. Copy .env.example 

```bash
copy .env.example .env
```
3. Generate Key

```bash
php artisan key:generate
```
4. Install Composer

```bash
composer install
```
5. Import database
6. Update .env value

## Complies Theme
1. Go to theme derectory 

```bash
cd themes/cartlooks-theme
```
2. Install Npm

```bash
npm install
```
3. Run Dev server

```bash
npm run watch
```
4. Complies in production

```bash
npm run production
```
